# ⚠️ КРИТИЧЕСКИ ВАЖНО: ПРОВЕРКА НА ФРОНТЕНДЕ СЕЙЧАС!

## Что нужно проверить:

### 1. Убедитесь, что все сервисы запущены:

```powershell
# Backend должен быть перезапущен (чтобы подхватить новый роутер)
# Frontend должен быть запущен
# Ollama Service должен быть запущен (опционально)
```

### 2. Откройте страницу деталей парсинга:

1. Откройте браузер: `http://localhost:3000`
2. Перейдите на страницу с результатами парсинга (любой завершенный run)
3. URL должен быть вида: `http://localhost:3000/parsing-runs/{runId}`

### 3. Проверьте чекбоксы:

- ✅ Должны быть видны чекбоксы слева от каждого домена в списке
- ✅ Кликните на чекбокс - домен должен выделиться
- ✅ Проверьте счетчик "Выбрано: X" вверху списка

### 4. Проверьте кнопки:

- ✅ Кнопка "Выбрать все" - должна выделить все домены
- ✅ Кнопка "Снять выбор" - должна снять выделение со всех доменов
- ✅ Кнопка "Копировать (X)" - должна появиться при выборе доменов
  - Кликните - домены должны скопироваться в буфер обмена
  - Должно появиться уведомление "Скопировано X доменов"

### 5. Проверьте извлечение ИНН:

- ✅ Выберите несколько доменов (чекбоксами)
- ✅ Кликните "Извлечь ИНН (X)"
- ✅ Должно открыться модальное окно с заголовком "Результаты извлечения ИНН"
- ✅ Должен отображаться прогресс: "Обработка: X из Y доменов"
- ✅ После завершения должны отобразиться результаты:
  - Для каждого домена: статус, ИНН (если найден)
  - Пруфы (контекст, метод, уверенность)

### 6. Проверьте пруфы (контекст):

- ✅ Для домена с найденным ИНН должны отобразиться:
  - URL, откуда взят ИНН (кликабельная ссылка)
  - Метод извлечения (regex/ollama)
  - Уровень уверенности (высокая/средняя/низкая)
  - Контекст - фрагмент текста вокруг ИНН

### 7. Проверьте консоль браузера (F12):

- ✅ Нет ошибок JavaScript
- ✅ Нет ошибок в Network tab
- ✅ Все запросы успешны (статус 200, 201, 204)

## Если что-то не работает:

1. **Endpoint возвращает 404:**
   - Перезапустите Backend: `cd backend && uvicorn app.main:app --reload`

2. **Ollama Service не запущен:**
   - Запустите: `cd ollama_inn_extractor && python run_api.py`

3. **Ошибки в консоли браузера:**
   - Проверьте логи Backend
   - Проверьте, что все импорты правильные
